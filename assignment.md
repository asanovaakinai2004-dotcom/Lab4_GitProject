# Лабораторная работа №4  
## Тема: Работа с Git и GitHub

---

## Цель работы
Освоить базовые навыки работы с системой контроля версий Git и удалённым репозиторием GitHub.

---

## Задание

1. Создать папку проекта **Lab4_GitProject**.
2. Создать внутри неё файл **assignment.md**.
3. Инициализировать локальный git-репозиторий.
4. Создать первый коммит.
5. Создать удалённый репозиторий на GitHub.
6. Связать локальный репозиторий с GitHub.
7. Отправить изменения (push) в удалённый репозиторий.
8. Вставить в `assignment.md` описание выполнения лабораторной работы.

---

## Выполнение работы

### 1. Создание проекта
Создана папка `Lab4_GitProject`.

### 2. Создание файла
Создан файл `assignment.md`.

### 3. Инициализация git
Команда:
git init

### 4. Добавление файлов и коммит
git add .
git commit -m "Первый коммит"

### 5. Создание GitHub-репозитория
На сайте GitHub создан репозиторий **Lab4_GitProject**.

### 6. Связывание локального и удалённого репозиториев
git remote add origin https://github.com/ВАШ_ЛОГИН/Lab4_GitProject.git

git branch -M main
git push -u origin main

### 7. Результат
Файлы успешно загружены на GitHub, репозиторий отображается онлайн.

---

## Вывод
В ходе работы освоены:
- создание локального git-репозитория,
- выполнение коммитов,
- создание удалённого репозитория,
- отправка данных на GitHub.
